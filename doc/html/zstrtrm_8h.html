char* zstrtrm ( char *  str)

Trim function.

This function removes the trailing and leading space characters f rom the string given with the argument str.

Parameters
strpointer to the input string

Example

#include <stdio.h>
#include <zstring.h>
int main() {
char s[]=" Free software is a matter of liberty. ";
printf("%sLike free speech!\\n",zstrtrm(s));
return 0;
}

Output

Free software is a matter of liberty.Like free speech!
Returns
zstrtrm function returns a pointer to the resulting string.
