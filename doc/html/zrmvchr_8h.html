char* zrmvchr ( char *  str,

)

Remove characters from a string.

All characters contained in the string argument b are removed from the string given with the argument str.

Parameters
strpointer to the input string
bstring containing characters to be removed

Example

#include <stdio.h>
#include <zstring.h>
int main() {
char s[]="this is a random string to test the function.";
char *b=" .";
printf("%s\\n",zrmvchr(s,b));
return 0;
}

zrmvchr function returns a pointer to the resulting string.
