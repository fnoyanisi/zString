int zstrchr ( const char *  str,

Locates the first occurance of a character in a string.

The function looks for the first occurance of the character given by the argument c in the string given by the argument str.

Parameters
strpointer to the input string
ccharacter to be located

Example

#include <stdio.h>
#include <zstring.h>
int main() {
char *t="zString is cool!";
printf("%d\n",zstrchr(t,'t'));
return 0;
}

Output

index = 1
Returns
zstrchr function returns the index of the first occurance of the character c inside the string str, or -1 if the character cannot be found.
