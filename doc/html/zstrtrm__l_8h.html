char* zstrtrm_l ( char *  str)

Left trim function.

This function removes the leading space characters from the string given with the argument str.

Parameters
strpointer to the input string

Example

#include <stdio.h>
#include <zstring.h>
int main() {
char s[]=" Free software is a matter of liberty. ";
printf("%sLike free speech!\\n",zstrtrm_l(s));
return 0;
}

Output

Free software is a matter of liberty. Like free speech!
Returns
zstrtrm_l function returns a pointer to the resulting string.
